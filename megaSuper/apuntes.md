# Introduccion a JS

Primero, decir que JavaScript es debilmente y dinamicamente tipado. Que significa esto?:

**Dinamicamente Tipado:** significa que un objeto puede, en un momento, estar apuntando 
a un string y luego a un objeto Persona por ejemplo. No restringe el tipo. Por ejemplo,
en Java, cuando definimos algo como int, sera int por el resto del programa.

Omito la explicacion de lo que se considera *debilmente tipado* debido a que es un concepto
algo difuso.
